Bienvenue dans la documentation du TIC-TAC-TOE :

Cette documentation est dédiée au projet TIC-TAC-TOE et sert de manuel de référence. Elle fournit des explications détaillées pour chaque classe et fonction utilisées dans le développement de ce jeu.

Bref descriptif du projet :

Ce projet de développement d’application s’inscrit dans le cadre de l’Unité d’Enseignement Génie Logiciel et Algorithmes de notre formation (S5). Ce projet vise à concevoir le classique TIC-TAC-TOE avec de nouvelles fonctionnalités, permettant à un joueur humain d'affronter un ordinateur (une AI) utilisant l’algorithme Minimax pour prendre des décisions optimales.

Comment faire fonctionner le projet :

Pour implementer le jeu "TIC-TAC-TOE" sur votre environnement, suivez ces étapes simples :

  1. Commencez par installer Python3 sur votre ordinateur.
  2. Ensuite installez les bibliothèques utilisées dans le dévloppement de l'application :
    • Biblithèque TKinter pour le devloppement de l'interface du jeu: Cette bibliothèque est généralement incluse avec l’installation de Python.
    • Bibliothèque Pillow pour le traitement des images : exécutez sur un terminal la commande :
      pip install Pillow
      Si cette commande ne fonctionne pas avec Python 3, essayez plutôt :
      pip3 install Pillow
  3. Lancez le jeu en exécutant le fichier main.py.
  4. En cas d’erreur d'affichage liée à l'image en arrière-plan : modifiez le chemin vers l'image en utilisant un chemin absolu.
